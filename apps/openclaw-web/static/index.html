多轮上下文文本、图片、文件统一接入同一会话，不再把附件当独立话题。
游客模式同一 IP 每小时最多 20 条消息，每天最多 3 张图片；3 小时后自动清理会话。
会话管理支持重命名、归档、折叠查看历史归档，并保留更自然的聊天布局。
双主题支持浅色 / 深色 / 跟随系统。主题切换位置已避开聊天区操作按钮。
